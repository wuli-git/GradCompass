============================================
厦门大学毕业生去向数据分析智能体
软件版本依赖说明
============================================

操作系统: Windows 11 (开发环境)
编程语言: Python 3.13.0

核心依赖版本:
  pandas              2.2.3
  numpy               2.0.2
  faker               33.1.0
  SQLite              3.x (Python内置sqlite3模块)
  sqlalchemy          2.0.36
  langchain           1.3.0
  langchain-community 0.4.2
  langchain-core      1.4.8
  langchain-openai    0.2.14
  openai              2.44.0
  streamlit           1.58.0
  plotly              6.8.0
  pyecharts           2.1.0
  python-dotenv       1.0.1
  openpyxl            3.1.5

大模型:
  DeepSeek-Chat (deepseek-chat)
  调用方式: DeepSeek API (OpenAI SDK 兼容)
  API地址: https://api.deepseek.com

安装命令:
  pip install pandas==2.2.3 numpy==2.0.2 faker==33.1.0
  pip install sqlalchemy==2.0.36
  pip install langchain==1.3.0 langchain-community==0.4.2 langchain-openai==0.2.14
  pip install streamlit==1.58.0 plotly==6.8.0 pyecharts==2.1.0
  pip install python-dotenv==1.0.1 openpyxl==3.1.5

或一键安装:
  pip install -r requirements.txt
